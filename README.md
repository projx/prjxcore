# prjxCore

## Install

`pip install https://github.com/projx/prjxcore/archive/refs/tags/v0.17.tar.gz`

`pip install  git+https://github.com/projx/prjxcore.git`

`pip install --upgrade git+https://github.com/projx/prjxcore.git`
